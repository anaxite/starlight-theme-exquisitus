## A theme built around the sentence

A lot of documentation is skimmed.
Why not make it more pleasant to read?
Exquisitus is composed for the long read: a hyperlegible body face, a serif that gives headings the cadence of a printed page, and a measure held to a comfortable line length so your eyes never lose their place.
