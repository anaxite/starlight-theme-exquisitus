Exquisitus is a Starlight [plugin](https://starlight.astro.build/reference/plugins/). You add it to an existing Starlight site; it brings its own fonts, palette, and component refinements.

<Steps>

1. **Install the theme** alongside Starlight.

   <Tabs syncKey="pkg">
   	<TabItem label="pnpm">
   	```sh
   	pnpm add starlight-theme-exquisitus
   	```
   	</TabItem>
   	<TabItem label="npm">
   	```sh
   	npm install starlight-theme-exquisitus
   	```
   	</TabItem>
   	<TabItem label="Bun">
   	```sh
   	bun add starlight-theme-exquisitus
   	```
   	</TabItem>
   </Tabs>

2. **Register the plugin** in your Starlight configuration.

   ```js title="astro.config.mjs" ins={[2, 8]}
   import starlight from '@astrojs/starlight';
   import starlightThemeExquisitus from 'starlight-theme-exquisitus';
   import { defineConfig } from 'astro/config';

   export default defineConfig({
     integrations: [
       starlight({
         plugins: [starlightThemeExquisitus()],
         title: 'My Docs',
       }),
     ],
   });
   ```

3. **Start the dev server** and read.

   ```sh
   pnpm dev
   ```

</Steps>

<Aside type="note" title="Search runs in production builds">
	Starlight's search is powered by [Pagefind](https://pagefind.app/), which indexes the site at
	build time — so the search box is inert under `pnpm dev` and shows a short placeholder instead of
	results. To try it, build and preview:

	```sh
	pnpm build && pnpm preview
	```
</Aside>

<Aside type="tip" title="Bring your own brand">
	Every colour is a CSS custom property. Override `--sl-color-accent` and friends in your own
	stylesheet to retune the palette without forking the theme.
</Aside>

## What the plugin does

Registering the plugin makes three changes to your site:

- **Self-hosts three typefaces** — Atkinson Hyperlegible Next, Spectral, and JetBrains Mono — via `@fontsource`, so nothing is fetched from a third-party origin.
- **Replaces the colour system** with an OKLCH palette for light and dark modes, layered so your own `customCss` still wins.
- **Refines the core components** — header, sidebars, asides, cards, code frames, and the hero — without changing Starlight's markup or behaviour.

<Aside type="note">
	Exquisitus never overrides a component you have already set yourself. If there is a conflict, it
	steps aside and logs a warning so your configuration stays in charge.
</Aside>
