## A theme built around the sentence

Most documentation is read, not skimmed. Exquisitus is composed for the long read: a hyperlegible body face, a serif that gives headings the cadence of a printed page, and a measure held to a comfortable line length so your eyes never lose their place.

<CardGrid>
	<Card title="Legible by design" icon="open-book">
		Body text is set in **Atkinson Hyperlegible Next**, a typeface engineered by the
		Braille Institute to keep letterforms distinct at every size.
	</Card>
	<Card title="Editorial structure" icon="document">
		Headings shift to **Spectral**, a screen-first serif, so documents read with the
		rhythm of print while the sans-serif body stays crisp.
	</Card>
	<Card title="Colour with intent" icon="seti:css">
		A warm honey-amber and a calm petrol-teal each do one job. Everything else is ink,
		paper, and breathing room — verified to WCAG&nbsp;2.2&nbsp;AA.
	</Card>
	<Card title="Yours to host" icon="rocket">
		All three faces are OFL-licensed and self-hosted. No third-party font request, no
		flash of unstyled text, no surprise dependency.
	</Card>
</CardGrid>
