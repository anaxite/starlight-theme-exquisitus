## A theme built around the sentence

A lot of documentation is skimmed.
Why not make it more pleasant to read?
Exquisitus is built for the long read: an editorial serif set at book size, a humanist sans that gives headings and chrome a voice of their own, and a measure held short enough that your eye returns cleanly to the next line.

## See for yourself

The showcase pages are plain Markdown rendered by the theme.
What you see is what your own documentation would look like.
